routes calls
	- /Invite
		- curl -X POST https://localhost/match/invite \
		-H "Content-Type: application/json" \
		-d '{"id":"1","game_type":"RANKED"}'

		- Success
			{
				"type": "INVITE_CREATED",
				"link": "https://localhost/match/join_party/<token>",
				"code": 200
			}

	-/join_party
		- use the link return of /invite
		- curl -X POST https://localhost/match/join_party/1dc22c2c573de370fbdd772f80692265 \
			-H "Content-Type: application/json" \
			-d '{"id":"2"}' -k
		- Success
			{
				"type": "JOINED_PARTY",
				"code": 200
			}
	
	-/leve_party
		- curl -X POST https://localhost/match/leave_party \
			-H "Content-Type: application/json" \
			-d '{"id":"1"}'
	-Success
		{type: 'LEFT_PARTY', code: 200}

	- /party
		curl -X GET "https://localhost/match/party?id=1" -H "Content-Type: application/json" -k

		- Success

				"type":"PARTY_INFO",
