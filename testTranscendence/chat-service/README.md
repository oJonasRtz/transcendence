**CHAT-SERVICE:** that service works as a proxy to connect the API-GATEWAY to the database using specifically functions to handle with public and private chats.

> **CONTROLLERS**

**storeMessage:** that function works storing the message to the public chat.

**getAllMessages:** that function works getting all public messages.

**getAllPrivateMessages:** that function works getting all private messages.

**storePrivateMessage:** that private message stores private messages
